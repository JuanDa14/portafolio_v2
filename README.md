## Portafolio_v2

### Descripción

Este es un portafolio que muestra mis habilidades y proyectos realizados.

### Tecnologías

- HTML
- CSS
- Tailwind
- TypeScript
- React
- Node
- Express
- MongoDB
- Mongoose

### Instalación

- Clonar el repositorio
- Instalar dependencias
- Ejecutar el servidor

### Ejecución

- Clonar el repositorio

```bash
git clone
```

- Instalar dependencias

```bash
npm install
```

- Ejecutar el servidor

```bash
npm run dev
```
